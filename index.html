UK Mortgage Broker Lead Finder

Finds leads, writes to a master Google Sheet automatically, deduplicates across every run, and tracks outreach status.

Search settings

Two APIs needed in Google Cloud Console (both free):
  1. Go to console.cloud.google.com → create a project
  2. Enable Places API and Maps JavaScript API
  3. Create an API key under Credentials and paste below
🔍

Enter your API key and click Run search to find leads

Google Sheets setup

One-time setup — creates your master leads sheet:
  1. Go to sheets.google.com and create a new blank spreadsheet
  2. Name it "UK Mortgage Leads Master"
  3. Copy the Sheet ID from the URL — it's the long string between /d/ and /edit
  4. Paste it below, then click Save
  5. Every time you run a search and click "Sync to Google Sheet", new leads are appended and duplicates are skipped automatically
To write to Google Sheets you also need to paste the Google Apps Script below into your sheet. Go to Extensions → Apps Script in your sheet, paste the script, and click Deploy.

The Web App URL comes from deploying the Apps Script below. It's what allows this tool to write directly to your sheet.

Apps Script — paste this into your Google Sheet

Gmail email sender script

How this works:
  • Paste this into a second Apps Script file in your Google Sheet
  • It reads every lead with status "Not Contacted" and an email address
  • Sends a personalised email using their company name, city and offer type
  • Updates the status to "Email Sent" and logs the date sent
  • Skips guessed emails by default (toggle below)
  • Sends max 50 per day automatically — spread across morning, midday and afternoon
  • Run it daily by setting a time-based trigger in Apps Script
Before sending: edit the YOUR_NAME, YOUR_WEBSITE and email templates in the script below to match your business.
